YouTube Shorts Auto-Scroller

Extension Settings

Button Position Reset
To reset the button position to its default location, open the browser console (F12) and run this command, then refresh the page:
localStorage.removeItem('yt-shorts-auto-scroll-btn-pos');
After running the command, refresh the YouTube Shorts page to see the button in its default position.
